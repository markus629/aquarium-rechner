Globale Pumpen-Geschwindigkeit✓ gespeichert

Gilt für alle vier Pumpen — direkt in Motor-Schritten, funktioniert auch bei unkalibrierten Pumpen (für den Kalibrier-Lauf selbst).

Pumpen-Kalibrierung

Der ESP fährt eine definierte Anzahl Schritte, du misst die geförderte Flüssigkeit mit einem Messbecher und gibst die ml ein. Daraus ergibt sich ml pro Motor-Schritt.

Kanister & Pumpen

Kapazität, aktueller Füllstand und Kalibrierung jeder Pumpe / jedes Kanisters.

pH-Sensor-Kalibrierung

Zweipunkt-Kalibrierung mit pH-4- und pH-7-Pufferlösung. Lege die Sonde in den jeweiligen Puffer, warte bis sich der Wert stabilisiert (~30 Sek.), klick Start. Der ESP sampelt 10 Sekunden lang und speichert die Voltage.

ESP einrichten

So verbindest du einen ESP32 mit deinem Account:

  1. Account-Zugang bereithalten (E-Mail + Passwort — bei Bedarf hier zurücksetzen):
    Lade Account-Info …
  2. ESP flashen — siehe README im GitHub-Repo
  3. ESP einschalten — beim ersten Start öffnet er ein WLAN namens AquariumRechner-Setup
  4. Mit dem WLAN verbinden — Browser öffnet automatisch das Setup-Portal (sonst http://192.168.4.1)
  5. WLAN-Daten + deine E-Mail + dein Passwort (dasselbe wie beim Web-Login) eintragen → Speichern
  6. ESP startet neu und meldet sich automatisch an. Status erscheint hier oben.

Firmware-Update (OTA via GitHub)

Der ESP holt sich Firmware-Updates direkt aus den GitHub-Releases. Kein Kreditkarten-gebundener Cloud-Storage nötig.

Lade Firmware-Info …

Eigene Firmware veröffentlichen: In GitHub → Releases → „Draft a new release" → Tag v1.0.4 → kompilierte .bin als Asset anhängen → „Publish release". ESP findet sie automatisch.

Ausfall-Benachrichtigung✓ gespeichert

ESP pingt diese URL bei jedem Heartbeat (alle 30 s). Bleibt der Ping aus, schickt der Dienst dir Email/Telegram/Discord/Pushover — sodass du merkst wenn etwas nicht stimmt.

📖 So richtest du healthchecks.io ein (kostenlos)
  1. Account auf healthchecks.io anlegen (kostenlos, 20 Checks frei)
  2. „Add Check" → Name: Aquarium-Rechner ESP
  3. Schedule: Simple Periodic, Period: 1 minute, Grace: 5 minutes (ESP pingt zwar alle 30 s, aber das Free-Tier verlangt mindestens 1 Minute — egal, mehr Pings sind erlaubt)
  4. Tab „Integrations" → Email aktivieren (oder Telegram/Discord/Pushover/…)
  5. Ping-URL (https://hc-ping.com/<uuid>) kopieren und oben einfügen
  6. Speichern → ESP übernimmt die URL beim nächsten Settings-Sync (max 2 min)

Backup & Wiederherstellung

Exportiert/importiert alle Daten dieses Aquariums (Settings, Pumpen, pH-Kalibrierung, Messungen, Dosierungen) als JSON-Datei.

Import überschreibt die aktuellen Daten in der Cloud. Empfohlen vorher Export!

Dieser Rechner dient als Hilfsmittel. Alle Empfehlungen ohne Gewähr — immer mit Augenmaß dosieren.
